# Which pin goes where?
## General Pins
| Arduino  | Sensor           |
|:--------:|:----------------:|
| A1       | Internal Temp    |
| A2       | External Temp    |
| A3       | Voltage Pin      |
| 3        | Buzzer           |
| 4        | External LM60 EN |
| 5        | Internal LM60 EN |
| 6        | SD Card          |
| 8        | GPS TX           |
